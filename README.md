# iWebp - Convert Images To Webp

Download [the latest release from macOS appstore](https://apps.apple.com/us/app/id1663442012)

V2.1
---
- Remove firebase analytics
- Bug fixed and Performance improvements.

V2.0.1
---
- Bug fixed and Performance improvements.

V2.0
---
- Support convert to .avif format
- Support copy image to clipboard not copy the filename

V1.1
---
- Support .gif animated images convert to animated webp

V1.0
---
- Easy batch convert images to .webp format
- Easy batch convert images to .avif format 
- Quick action copy image not image file name to system clipboard then quick paste to other application
- Keep privacy, No need to upload any images

Usage: 

1. Select image(s) 
2. Right context menu -> Create webp 
3. Select target quailty
4. Done
